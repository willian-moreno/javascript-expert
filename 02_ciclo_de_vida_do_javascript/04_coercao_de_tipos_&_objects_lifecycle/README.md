### Coerção de Tipos & Objects lifecycle: toString, valueOf e Symbol.toPrimitive
1. Coerção de tipos, ou conversão e tipos, é um conceito utilizado para explicar os comportamentos da linguagem.
2. A coerção de tipos é um processo de conversão de um valor para qualquer outro tipo.
3. Em JavaScript qualquer tipo de dado está sujeito a coerção.
4. Só existem 3 tipos de conversão no fim: String, Boolean, Number.
5. Existem 2 tipos de coerção: a coerção implícita e a coerção explícita.
6. A coerção implícita é aquela geralmente utilizada com operadores (soma de string com number, if usando uma variável na condição que não é booleana, etc.).
7. É importante utilizar o ``===`` ao invés do ``==`` em comparações, para evitar a coerção do dado, que pode gerar problemas.
